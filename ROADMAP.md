# Tagalog Learning Platform - Technical Roadmap

This roadmap outlines the next technical priorities to enhance the educational value of the platform, with a focus on 
implementing support for Tagalog's language focus system and adding essential learning components.

## Priority 1: Implement 1st lesson "Alphabet & Pronunciation"

**Objective:** Create a comprehensive static structure to support Tagalog's "Alphabet & Pronunciation" introductory lesson.
- It will be the first lesson of the unit.
- It will introduce the Tagalog alphabet to new students.
- It will contain phonemes and samples

**Tasks:**
- Design and implement the `Lesson` interface with focus, aspect, and conjugation data
- Define the content for the first lesson "Alphabet & Pronunciation".
